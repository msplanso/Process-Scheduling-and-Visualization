INSTRUCTIONS FOR VISUALIZATION OF THE SCHEDULING PROCESS

INSTRUCTIONS

  • The objective of this GUI is to graphically visualize of the different scheduling process.
  • The user can create a scheduling process by dragging and dropping individual Steps from Scheduling Models to the GANT chart.
  • After creating a scheduling process, the user can see the visualization by clicking the start button.
  • The operation can be paused in the middle by clicking the Stop button and changes can be made to the schedule
  • If changes are made in the middle the process has to start from the beginning again
  • Only two processes can start at the same time.
  • Starting more than two processes at the same time is infeasible.
  • The GANT chart picturizes the execution of the process and also the elapsed time
  • The recipes and steps can be chosen in random order.
  • The user can select different Scheduling Model from the Type button.

GANT CHART TERMINOLOGIES

  • If a recipe is dragged and dropped in the chart it will be picturized as a block in the respective stages.
  • To move the block left mouse click can be used.
  • To delete a block right mouse click can be used.
  • The schedule can be changed both before and during the plant operation
